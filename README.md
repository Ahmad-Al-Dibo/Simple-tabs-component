# Solo Leveling Tabs Component
![Voorbeeld Pagina](voorbeeld-pagina.png)
Een simpele tabs component gemaakt met HTML, CSS en JavaScript. De pagina gebruikt een Solo Leveling anime-frame als achtergrond en toont per tab passende content over Sung Jinwoo, het System en de subway instance dungeon.

## Features

- Vier klikbare tabs: `System`, `Dungeon`, `Monsters` en `Level Up`
- Actieve tab krijgt een paars neon-effect
- Content verandert zonder de pagina te herladen
- Achtergrondafbeelding vult het hele scherm
- Donkere content-box zodat de tekst goed leesbaar blijft

## Gebruikte technieken

- HTML voor de structuur
- CSS voor layout, achtergrond en styling
- JavaScript voor DOM-manipulatie en click events

## Bestanden

- `index.html` - de tabs en content van de pagina
- `styles.css` - de styling van de pagina
- `script.js` - de logica waarmee de tabs werken
- `thumb-1920-1347905.png` - de actieve achtergrondafbeelding

## Project openen

Open `index.html` direct in je browser.

Je kunt ook een lokale server gebruiken, bijvoorbeeld met VS Code Live Server.

## Leerdoel

Dit project oefent basisvaardigheden in JavaScript:

- Elementen selecteren met `querySelector` en `querySelectorAll`
- Classes toevoegen en verwijderen
- Content tonen en verbergen
- Reageren op click events
